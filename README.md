# dotfiles // brenno A

Setup automatizado do macOS com Homebrew, apps essenciais e configuração do AeroSpace.

---

## 🚀 Workflow

1. **Homebrew**
   Verifica, instala e atualiza o Homebrew.

2. **Apps**
   Instala automaticamente todos os aplicativos definidos no script.

3. **AeroSpace**
   Instala o window manager e aplica a configuração personalizada.

4. **Limpeza**
   Remove arquivos temporários e dependências não utilizadas.

---
